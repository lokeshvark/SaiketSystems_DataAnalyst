# 📊 Customer Churn Analysis & Predictive Modeling

## 🚀 Project Overview

This repository contains a comprehensive Customer Churn Analysis project completed as part of the Data Analyst Internship. The project focuses on analyzing customer behavior, identifying churn patterns, and building predictive models to help businesses improve customer retention.

Using the Telco Customer Churn dataset, various data analysis, visualization, and machine learning techniques were applied to uncover key insights and predict customers who are likely to leave the service.

---

## 🎯 Project Objectives

* Clean and preprocess customer data
* Perform Exploratory Data Analysis (EDA)
* Identify factors influencing customer churn
* Visualize customer behavior and churn trends
* Build and evaluate churn prediction models
* Generate actionable business recommendations

---

## 📊 Key Insights

✅ Customers with higher monthly charges tend to show higher churn rates.

✅ Contract type significantly impacts customer retention.

✅ Long-term customers are less likely to churn.

✅ Electronic payment methods are associated with higher churn probability.

✅ Internet and support-related services influence customer loyalty.
